####依賴

```
{
	"express":"3.1.1",
	"jade":"*",
	"mongodb":"*",
	"connect-mongo":"*",
	"canvas":"*",
	"cron":"*"
}
```

當然，通過`npm install`就可以完成了。

她上傳頭像的裁取的時候，使用的是canvas，將圖片轉為 `data uri` 來向後臺傳輸，nodejs再將圖像用canvas畫出來（其實當做字段存入`mongodb`也可以，不過性能上沒有測試，以後會補上兩種方案測試的結果）。
